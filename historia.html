Historia

  1. ¿Qué es GNU/Linux?

    2. GNU/Linux es un sistema operativo libre tipo Unix compuesto por software libre y de código abierto. El sistema es la combinación de varios proyectos, entre los cuales destacan GNU (encabezado por Richard Stallman y la Free Software Foundation) y el núcleo Linux (encabezado por Linus Torvalds). El nombre preciso es GNU/Linux, pero Linux se usa con más frecuencia.

  2. Richard Stallman y el movimiento del software libre.

    3. Richard Stallman es un físico, programador estadounidense y activista que fundó el movimiento del software libre, el sistema operativo GNU y la Free Software Foundation (Fundación para el Software Libre). En 1983 inició el proyecto GNU que 12 años después derivó en la aparición del sistema operativo libre y gratuito, ahora popularmente conocido como Linux. El movimiento del software libre ha tenido un impacto significativo en la industria del software y ha contribuido a lademocratización de la tecnología.

  3. ¿Qué es el software libre?

    4. El software libre es un término que se utiliza para describir un tipo de software que se puede usar, copiar, modificar y redistribuir libremente. El software libre se diferencia del software propietario en que el código fuente del software libre está disponible para su modificación y redistribución por cualquier persona, mientras que el código fuente del software propietario está disponible solo para su uso por parte de los usuarios finales.

  4. Linus Torvalds y el proyecto Minix.

    5. Linus Torvalds es un ingeniero de software conocido por iniciar y mantener el desarrollo del kernel Linux. El kernel Linux se basó en el sistema operativo libre Minix y en algunas herramientas, varias utilidades y los compiladores desarrollados por el proyecto GNU.

    MINIX es un clon del sistema operativo Unix distribuido junto con su código fuente. Fue creado con propósitos pedagógicos y su código está a disposición del público para ser estudiado y modificado a voluntad en cada una de sus partes o capas.

  5. ¿Por qué Linux es importante?

    6. Linux es importante es porque es un sistema operativo libre y de código abierto. Esto significa que cualquier persona puede ver, modificar y distribuir el código gran cantidad de colaboración y desarrollo en la comunidad de Linux.

    Otra razón por la que Linux es importante es porque se utiliza ampliamente en servidores, supercomputadoras y otros sistemas críticos. También se utiliza en dispositivos móviles como teléfonos inteligentes y tabletas a través de sistemas operativos basados ​​en Linux como Android.

  6. El surgimiento de las distribuciones de Linux

    7. Una distribución de Linux es un sistema operativo instalable diseñado a partir del kernel de Linux, el cual admite programas de usuarios y bibliotecas.

    En la primera mitad de la década de los 90 vivimos el desarrollo de las primeras distribuciones de Linux de la mano de Slackware, Debian y Red Hat. Hoy en día hay muchas distribuciones disponibles y cada una tiene sus propias características únicas y público objetivo.
